• Filter the rules with the 'Mod' and 'Category' dropdowns
  • Set a rule value in the right most column
  • Reset a rule to its default value with the button in the middle column
  • Reset all settings to default with the 'Reset All' button at the top
  • See which mod and category a rule belongs to and its description by hovering over the rule name
  • Export a 'carpet.conf' file via the 'Export' button at the top. Place that in your world's save folder
